# KaloriSayar Infrastructure

Infrastructure assets for local development and production deployment templates.

## Docker Compose

Located in `infra/docker/docker-compose.yml`, providing Postgres, Redis, MinIO, and PostHog for local testing.

## Terraform

`infra/terraform` contains a placeholder for a single-VM deployment. Populate with modules targeting your preferred cloud provider (e.g., AWS Lightsail, DigitalOcean Droplet).
