# KaloriSayar Admin

This Next.js project powers the moderation and operations dashboard. The current scaffold demonstrates queue summaries using shared UI primitives.

## Scripts

- `pnpm dev --filter @kalorisayar/admin` – Run on http://localhost:3001.
- `pnpm build --filter @kalorisayar/admin` – Build production bundle.
- `pnpm lint --filter @kalorisayar/admin` – Run linting.
