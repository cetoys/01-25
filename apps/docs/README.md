# KaloriSayar Documentation

Docusaurus project describing the architecture, runbooks, and API reference for KaloriSayar.

## Scripts

- `pnpm start --filter @kalorisayar/docs` – Local development server.
- `pnpm build --filter @kalorisayar/docs` – Build static documentation site.

## Content Guidelines

- Maintain bilingual sections (English/Turkish) for user-facing operations guidance.
- Store diagrams using Markdown with Mermaid for easy diffing.
