# KaloriSayar Mobile (Expo)

This Expo Router project provides the cross-platform mobile experience. The current scaffold implements a bilingual dashboard screen and shared navigation layout.

## Scripts

- `pnpm start --filter @kalorisayar/mobile` – Launch Expo.
- `pnpm android --filter @kalorisayar/mobile` – Open Android simulator/device.
- `pnpm ios --filter @kalorisayar/mobile` – Open iOS simulator/device.
